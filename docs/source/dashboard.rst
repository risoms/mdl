Dashboard
"""""""""

.. raw:: html

   <div class="col-md-9">

To view the dashboard you'll need to download the mdl-dashboard standalone app directly from us at 
`https://mdl.psy.utexas.edu/mdl/dashboard <https://mdl.psy.utexas.edu/mdl/dashboard>`_.

Or install into your local version of python using ``pip``::

    pip install git+https://github.com/risoms/mdl-dashboard.git

Another option would be to to clone the `github <https://github.com/risoms/mdl-dashboard>`_ repository and install from your local copy::

    pip install .
